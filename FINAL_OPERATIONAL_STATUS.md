## Monitoring Dependency Resolution

### The Problem
```
Import Chain:
enhanced_analysis.py
  ‚Üí PDFFormulaEngine  
    ‚Üí ContentQualityAnalyzer
      ‚Üí performance_monitor
        ‚Üí monitoring_v3 (from google.cloud.monitoring)
          ‚Üí ‚ùå NOT in requirements-minimal.txt
```

### The Solution
Made monitoring optional in 3 files using try-except pattern:

**Pattern Applied**:
```python
try:
    from ...middleware.monitoring import performance_monitor
    MONITORING_AVAILABLE = True
except ImportError:
    # No-op decorator fallback
    def performance_monitor(operation_name):
        def decorator(func):
            return func
        return decorator
    MONITORING_AVAILABLE = False
```

**Files Fixed**:
1. `pdf_formula_engine.py` ‚úÖ
2. `action_layer_calculator.py` ‚úÖ
3. `content_quality_analyzer.py` ‚úÖ

**Result**: Engines work WITHOUT google-cloud-monitoring, but monitoring automatically enabled when dependency present.
